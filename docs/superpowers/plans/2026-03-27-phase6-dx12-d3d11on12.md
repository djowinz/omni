# Phase 6: DX12 Hook + D3D11On12 Renderer

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Make the overlay work on DX12 games by detecting the graphics API at runtime and using D3D11On12 to wrap DX12 back buffers into a D3D11-compatible surface that the existing D2D renderer can draw on.

**Architecture:** The existing Present/Present1/ResizeBuffers hooks already fire for DX12 games (they share the `IDXGISwapChain` vtable). The renderer gains a DX12 code path: on first frame, it queries the swap chain's device — if it's `ID3D12Device`, it captures the command queue via a hooked `CreateSwapChainForHwnd`, creates a `D3D11On12` device, and wraps the back buffer each frame. The D2D rendering code is unchanged — it always renders to a `ID2D1RenderTarget` from a DXGI surface regardless of the underlying API.

**Tech Stack:** Rust, `windows` crate 0.58 (D3D12, D3D11On12, DXGI), `minhook` for the additional `CreateSwapChainForHwnd` hook.

**Testing notes:** DX11 regression tested on existing DX11 game. DX12 tested on a DX12 game. Both should show the CPU usage overlay. No new unit tests — this is entirely Win32 API integration code.

**Depends on:** Phase 5 complete (D2D renderer, shared memory IPC, sensor polling all working).

```rust
// We hook that call to capture the queue for D3D11On12.

/// Captured DX12 command queue, set by the CreateSwapChainForHwnd hook.
pub static mut CAPTURED_COMMAND_QUEUE: Option<windows::Win32::Graphics::Direct3D12::ID3D12CommandQueue> = None;

/// IDXGIFactory2 vtable index for CreateSwapChainForHwnd.
/// IDXGIFactory2 inherits: IUnknown(3) + IDXGIObject(4) + IDXGIFactory(7) + IDXGIFactory1(2) = 16
/// IDXGIFactory2 own methods: 0=IsWindowedStereoEnabled, 1=CreateSwapChainForHwnd
pub const CREATE_SWAP_CHAIN_FOR_HWND_VTABLE_INDEX: usize = 15;

type CreateSwapChainForHwndFn = unsafe extern "system" fn(
    this: *mut c_void,          // IDXGIFactory2*
    p_device: *mut c_void,      // IUnknown* (the D3D device or command queue)
    hwnd: HWND,                 // target window
    p_desc: *const c_void,      // DXGI_SWAP_CHAIN_DESC1*
    p_fullscreen_desc: *const c_void, // DXGI_SWAP_CHAIN_FULLSCREEN_DESC* (nullable)
    p_restrict_to_output: *const c_void, // IDXGIOutput* (nullable)
    pp_swap_chain: *mut *mut c_void, // IDXGISwapChain1** out
) -> windows::core::HRESULT;

static mut ORIGINAL_CREATE_SWAP_CHAIN_FOR_HWND: Option<CreateSwapChainForHwndFn> = None;
```

```rust
// ─── CreateSwapChainForHwnd hook ─────────────────────────────────────────────

unsafe extern "system" fn hooked_create_swap_chain_for_hwnd(
    this: *mut c_void,
    p_device: *mut c_void,
    hwnd: HWND,
    p_desc: *const c_void,
    p_fullscreen_desc: *const c_void,
    p_restrict_to_output: *const c_void,
    pp_swap_chain: *mut *mut c_void,
) -> windows::core::HRESULT {
    // Try to QueryInterface the device parameter for ID3D12CommandQueue.
    // In DX12, CreateSwapChainForHwnd receives the command queue (not the device).
    // In DX11, it receives the D3D11 device — QI for command queue will fail, which is fine.
    use windows::core::Interface;
    use windows::Win32::Graphics::Direct3D12::ID3D12CommandQueue;

    let unknown: &windows::core::IUnknown = &*(p_device as *const windows::core::IUnknown);
    if let Ok(queue) = unknown.cast::<ID3D12CommandQueue>() {
        crate::logging::log_to_file("[hook] captured DX12 command queue from CreateSwapChainForHwnd");
        CAPTURED_COMMAND_QUEUE = Some(queue);
    }

    // Call the original
    if let Some(original) = ORIGINAL_CREATE_SWAP_CHAIN_FOR_HWND {
        original(this, p_device, hwnd, p_desc, p_fullscreen_desc, p_restrict_to_output, pp_swap_chain)
    } else {
        windows::core::HRESULT(0)
    }
}
```

```rust
/// Discover the IDXGIFactory2::CreateSwapChainForHwnd vtable address.
/// Creates a temporary D3D11 device to get the factory, reads the vtable.
unsafe fn discover_factory2_vtable() -> Result<*const c_void, String> {
    // Create a temporary D3D11 device to get a DXGI factory
    let mut device: Option<ID3D11Device> = None;
    D3D11CreateDevice(
        None,
        D3D_DRIVER_TYPE_HARDWARE,
        None,
        D3D11_CREATE_DEVICE_FLAG(0),
        None,
        D3D11_SDK_VERSION,
        Some(&mut device),
        None,
        None,
    )
    .map_err(|e| format!("D3D11CreateDevice (for factory vtable) failed: {e}"))?;

    let device = device.ok_or("D3D11CreateDevice returned null device")?;

    let dxgi_device: IDXGIDevice = device
        .cast()
        .map_err(|e| format!("device.cast::<IDXGIDevice>(): {e}"))?;

    let adapter = dxgi_device
        .GetAdapter()
        .map_err(|e| format!("GetAdapter: {e}"))?;

    let factory: IDXGIFactory2 = adapter
        .GetParent()
        .map_err(|e| format!("GetParent::<IDXGIFactory2>(): {e}"))?;

    // Read CreateSwapChainForHwnd from the vtable
    let raw_ptr: *mut *const *const c_void = Interface::as_raw(&factory) as _;
    let vtable: *const *const c_void = *raw_ptr;
    let create_sc_addr = *vtable.add(CREATE_SWAP_CHAIN_FOR_HWND_VTABLE_INDEX);

    crate::logging::log_to_file(&format!(
        "[hook] IDXGIFactory2::CreateSwapChainForHwnd at {:p}",
        create_sc_addr,
    ));

    Ok(create_sc_addr)
}
```

```rust
    /// Ensure we have a render target for the current swap chain back buffer.
    /// Detects DX11 vs DX12 on first call and uses the appropriate path.
    unsafe fn ensure_render_target(&mut self, swap_chain_ptr: *mut c_void) -> Result<(), String> {
        if self.render_target.is_some() {
            return Ok(());
        }

        let sc: IDXGISwapChain = std::mem::transmute_copy(&swap_chain_ptr);
        let sc = ManuallyDrop::new(sc);

        // Detect API on first call
        if self.api == GraphicsApi::Unknown {
            if sc.GetDevice::<ID3D11Device>().is_ok() {
                log_to_file("[renderer] detected DX11 swap chain");
                self.api = GraphicsApi::DX11;
            } else if sc.GetDevice::<ID3D12Device>().is_ok() {
                log_to_file("[renderer] detected DX12 swap chain — using D3D11On12");
                self.api = GraphicsApi::DX12;
            } else {
                return Err("Could not determine graphics API from swap chain".into());
            }
        }

        match self.api {
            GraphicsApi::DX11 => self.ensure_render_target_dx11(&sc),
            GraphicsApi::DX12 => self.ensure_render_target_dx12(&sc),
            GraphicsApi::Unknown => Err("Graphics API not detected".into()),
        }
    }

    /// DX11 path: get DXGI surface directly from back buffer.
    unsafe fn ensure_render_target_dx11(&mut self, sc: &ManuallyDrop<IDXGISwapChain>) -> Result<(), String> {
        let back_buffer: windows::Win32::Graphics::Dxgi::IDXGISurface = sc
            .GetBuffer(0)
            .map_err(|e| format!("GetBuffer::<IDXGISurface>(0) failed: {e}"))?;

        self.create_d2d_render_target(&back_buffer)
    }

    /// DX12 path: create D3D11On12 device, wrap back buffer, get DXGI surface.
    unsafe fn ensure_render_target_dx12(&mut self, sc: &ManuallyDrop<IDXGISwapChain>) -> Result<(), String> {
        // Create D3D11On12 device if we don't have one yet
        if self.d3d11on12_device.is_none() {
            self.create_d3d11on12_device(sc)?;
        }

        // Get the DX12 back buffer and wrap it as a D3D11 resource
        let back_buffer_dx12: windows::Win32::Graphics::Direct3D12::ID3D12Resource = sc
            .GetBuffer(0)
            .map_err(|e| format!("GetBuffer::<ID3D12Resource>(0) failed: {e}"))?;

        let d3d11on12 = self.d3d11on12_device.as_ref()
            .ok_or("D3D11On12 device not initialized")?;

        let flags = D3D11_RESOURCE_FLAGS {
            BindFlags: D3D11_BIND_RENDER_TARGET.0 as u32,
            ..Default::default()
        };

        let mut wrapped: Option<ID3D11Resource> = None;
        d3d11on12.CreateWrappedResource(
            &back_buffer_dx12,
            &flags,
            windows::Win32::Graphics::Direct3D12::D3D12_RESOURCE_STATE_RENDER_TARGET,
            windows::Win32::Graphics::Direct3D12::D3D12_RESOURCE_STATE_PRESENT,
            &mut wrapped,
        )
        .map_err(|e| format!("CreateWrappedResource failed: {e}"))?;

        let wrapped = wrapped.ok_or("CreateWrappedResource returned None")?;

        // Get DXGI surface from the wrapped D3D11 resource
        let surface: windows::Win32::Graphics::Dxgi::IDXGISurface = wrapped
            .cast()
            .map_err(|e| format!("cast wrapped resource to IDXGISurface: {e}"))?;

        self.wrapped_back_buffer = Some(wrapped);

        self.create_d2d_render_target(&surface)
    }

    /// Create the D3D11On12 device from the swap chain's DX12 device.
    unsafe fn create_d3d11on12_device(&mut self, sc: &ManuallyDrop<IDXGISwapChain>) -> Result<(), String> {
        let dx12_device: ID3D12Device = sc
            .GetDevice()
            .map_err(|e| format!("GetDevice::<ID3D12Device>() failed: {e}"))?;

        // Get the command queue captured by our CreateSwapChainForHwnd hook
        let command_queue = crate::hook::CAPTURED_COMMAND_QUEUE.as_ref()
            .ok_or("DX12 command queue not captured — CreateSwapChainForHwnd hook may not have fired")?;

        let queues: [Option<IUnknown>; 1] = [Some(command_queue.cast()?)];

        let mut d3d11_device: Option<ID3D11Device> = None;

        D3D11On12CreateDevice(
            &dx12_device,
            D3D11_CREATE_DEVICE_BGRA_SUPPORT.0 as u32,
            None, // no feature levels — use default
            Some(&queues),
            0, // node mask
            Some(&mut d3d11_device),
            None, // don't need immediate context
            None, // don't need feature level out
        )
        .map_err(|e| format!("D3D11On12CreateDevice failed: {e}"))?;

        let d3d11_device = d3d11_device.ok_or("D3D11On12CreateDevice returned null device")?;

        let d3d11on12: ID3D11On12Device = d3d11_device
            .cast()
            .map_err(|e| format!("cast to ID3D11On12Device: {e}"))?;

        self.d3d11on12_device = Some(d3d11on12);
        log_to_file("[renderer] D3D11On12 device created successfully");

        Ok(())
    }

    /// Shared helper: create a D2D render target from a DXGI surface.
    unsafe fn create_d2d_render_target(
        &mut self,
        surface: &windows::Win32::Graphics::Dxgi::IDXGISurface,
    ) -> Result<(), String> {
        let props = D2D1_RENDER_TARGET_PROPERTIES {
            r#type: windows::Win32::Graphics::Direct2D::D2D1_RENDER_TARGET_TYPE_DEFAULT,
            pixelFormat: D2D1_PIXEL_FORMAT {
                format: DXGI_FORMAT_UNKNOWN,
                alphaMode: D2D1_ALPHA_MODE_PREMULTIPLIED,
            },
            dpiX: 0.0,
            dpiY: 0.0,
            usage: windows::Win32::Graphics::Direct2D::D2D1_RENDER_TARGET_USAGE_NONE,
            minLevel: windows::Win32::Graphics::Direct2D::D2D1_FEATURE_LEVEL_DEFAULT,
        };

        let rt = self.d2d_factory
            .CreateDxgiSurfaceRenderTarget(surface, &props)
            .map_err(|e| format!("CreateDxgiSurfaceRenderTarget failed: {e}"))?;

        self.render_target = Some(rt);
        log_to_file("[renderer] D2D render target created");
        Ok(())
    }
```

```rust
    /// For DX12: acquire the wrapped back buffer before D2D rendering.
    /// Must be called before BeginDraw. No-op for DX11.
    pub unsafe fn acquire_wrapped_resource(&self) {
        if let (Some(d3d11on12), Some(wrapped)) = (&self.d3d11on12_device, &self.wrapped_back_buffer) {
            let resources: [Option<ID3D11Resource>; 1] = [Some(wrapped.clone())];
            d3d11on12.AcquireWrappedResources(&resources);
        }
    }

    /// For DX12: release the wrapped back buffer after D2D rendering.
    /// Must be called after EndDraw. No-op for DX11.
    pub unsafe fn release_wrapped_resource(&self) {
        if let (Some(d3d11on12), Some(wrapped)) = (&self.d3d11on12_device, &self.wrapped_back_buffer) {
            let resources: [Option<ID3D11Resource>; 1] = [Some(wrapped.clone())];
            d3d11on12.ReleaseWrappedResources(&resources);

            // Flush the D3D11 immediate context to submit the D2D work to the DX12 queue
            let d3d11_device: ID3D11Device = d3d11on12.cast().unwrap();
            let context = d3d11_device.GetImmediateContext().unwrap();
            context.Flush();
        }
    }
```

```rust
    /// Render a list of computed widgets onto the swap chain back buffer.
    pub unsafe fn render(&mut self, swap_chain_ptr: *mut c_void, widgets: &[ComputedWidget]) {
        if let Err(e) = self.ensure_render_target(swap_chain_ptr) {
            log_to_file(&format!("[renderer] failed to create render target: {e}"));
            return;
        }

        let rt = match &self.render_target {
            Some(rt) => rt,
            None => return,
        };

        // DX12: acquire the wrapped back buffer
        self.acquire_wrapped_resource();

        rt.BeginDraw();

        for widget in widgets {
            if widget.opacity <= 0.0 {
                continue;
            }

            let rect = D2D_RECT_F {
                left: widget.x,
                top: widget.y,
                right: widget.x + widget.width,
                bottom: widget.y + widget.height,
            };

            // Draw background
            let bg = &widget.bg_color_rgba;
            if bg[3] > 0 {
                let bg_color = D2D1_COLOR_F {
                    r: bg[0] as f32 / 255.0,
                    g: bg[1] as f32 / 255.0,
                    b: bg[2] as f32 / 255.0,
                    a: (bg[3] as f32 / 255.0) * widget.opacity,
                };

                if let Ok(brush) = rt.CreateSolidColorBrush(&bg_color, None) {
                    let radius = widget.border_radius[0];
                    if radius > 0.0 {
                        let rounded = D2D1_ROUNDED_RECT {
                            rect,
                            radiusX: radius,
                            radiusY: radius,
                        };
                        rt.FillRoundedRectangle(&rounded, &brush);
                    } else {
                        rt.FillRectangle(&rect, &brush);
                    }
                }
            }

            // Draw text
            let text = read_fixed_str(&widget.format_pattern);
            if text.is_empty() {
                continue;
            }

            let font_weight = if widget.font_weight >= 700 {
                DWRITE_FONT_WEIGHT_BOLD
            } else {
                DWRITE_FONT_WEIGHT_NORMAL
            };

            let text_format = self.dwrite_factory.CreateTextFormat(
                w!("Segoe UI"),
                None,
                font_weight,
                DWRITE_FONT_STYLE_NORMAL,
                DWRITE_FONT_STRETCH_NORMAL,
                widget.font_size,
                w!("en-us"),
            );

            let text_format = match text_format {
                Ok(tf) => tf,
                Err(_) => continue,
            };

            let _ = text_format.SetParagraphAlignment(DWRITE_PARAGRAPH_ALIGNMENT_CENTER);
            let _ = text_format.SetTextAlignment(DWRITE_TEXT_ALIGNMENT_CENTER);

            let fg = &widget.color_rgba;
            let fg_color = D2D1_COLOR_F {
                r: fg[0] as f32 / 255.0,
                g: fg[1] as f32 / 255.0,
                b: fg[2] as f32 / 255.0,
                a: (fg[3] as f32 / 255.0) * widget.opacity,
            };

            if let Ok(brush) = rt.CreateSolidColorBrush(&fg_color, None) {
                let text_wide: Vec<u16> = text.encode_utf16().collect();
                rt.DrawText(
```
